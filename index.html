Votos conselho tutelar

Sistema desenvolvido para a realização da votação do conselho tutelar no ano de 2019. O sistema utilizado conceitos de programação básicos, desenvolvido apenas com html, css e Js. Como se trata de uma aplicação de votação simples não há utilização de banco de dados externo, e o armazenamento de dados utiliza LocalStorage.
